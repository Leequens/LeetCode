# 问题分析：
给定一个整数 n，返回 n! 结果尾数中零的数量。

# 编程实现：
```C++
class Solution {
public:
    int trailingZeroes(int n) 
    {
        int res=0;
        while(n)
        {
            res=res+n/5;
            n=n/5;
        }
        return res;
    }
};
```
# 总结体会：

乍一看此题很简单，计算出n的阶乘然后统计尾数的0就好了，但是不论使用long还是long long最后通过oj时结果都会溢出。所以要使用其他方法，尾数多少个0就是看相乘时因式中有多少个10，10=2*5,。2有很多，4,6,8等都可以分解出2来，所以只要看能分解出多少个5就可以判断出尾数中有多少个0了。
